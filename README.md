## Getting Started

Clone this repository to your local machine to get started with PySide6 basic projects.

```bash
git clone https://github.com/aaleshpatil22/PySide6_Basic.git
```

Make sure you have Python and PySide6 installed.

```bash
pip install PySide6
```
